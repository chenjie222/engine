# Quant1X-Engine

量化交易（数据类）引擎 - 支持日K线、5分钟K线等多周期数据采集

## 1. 设计原则
- 更新历史数据，盘后更新历史数据
- 回补历史数据，对于新增特征组合或者因子要能回补历史数据，以便回测
- 盘中更新数据，盘中决策很重要，特征组合要有根据5档行情即时数据进行增量计算的能力
- 缓存数据必须具备按照日期切换的功能

## 2. 模块划分

| 级别 | 模块       | 功能        | 盘中更新数据 | 更新当日数据 | 回补历史数据 |
|:---|:---------|:----------|:-------|:-------|:-------|
| 0  | cache    | 数据缓存      | [ √ ]  | [ √ ]  | [ √ ]  |
| 0  | datasets | 数据集, 基础数据 | [ X ]  | [ √ ]  | [ √ ]  |
| 0  | factors  | 量化因子      | [ X ]  | [ √ ]  | [ √ ]  |
| 0  | features | 特征        | [ X ]  | [ √ ]  | [ √ ]  |
| 0  | tracker  | 回测        | [ X ]  | [ X ]  | [ X ]  |

## 3. 使用示例

### 3.1 更新数据

```shell
engine update --all
```

### 3.2 补登历史特征数据

```shell
engine repair --features=history --start=20230101
```

### 3.3 执行1号策略

```shell
engine --strategy=1
```

### 3.4 下载5分钟K线数据

#### 功能说明
下载通达信5分钟K线数据，支持简化CSV格式（7个字段），可指定日期范围，支持增量更新。

#### CSV格式
```csv
timestamps,open,high,low,close,volume,amount
2024-03-04 09:35:00,13.410000,13.530000,13.310000,13.330000,-15900905.994984,18332092.000000
2024-03-04 09:40:00,13.330000,13.410000,13.310000,13.360000,-9235373.394594,10653222.000000
```

#### 命令参数

| 参数 | 说明 | 示例值 |
|:---|:---|:---|
| `--code` | 指定股票代码 | `sz300773` |
| `--sector` | 指定板块 | `sh600`、`sz300`、`sh688` |
| `--all` | 下载所有股票 | - |
| `--start` | 开始日期（YYYYMMDD） | `20200101`、`20230101` |
| `--update` | 增量更新模式 | - |
| `--stats` | 查看统计信息（配合`--code`） | - |
| `--path` | 显示存储路径 | - |

#### 使用示例

**1. 下载单个股票（默认从2020-01-01开始）**
```shell
engine 5min --code=sz300773
```

**2. 从指定日期下载**
```shell
engine 5min --code=sz300773 --start=20240101
```

**3. 增量更新（只下载新增数据）**
```shell
# 第一次下载
engine 5min --code=sz300773 --start=20240101

# 之后只需增量更新
engine 5min --code=sz300773 --update
```

**4. 下载指定板块**
```shell
# 下载上海主板全部股票
engine 5min --sector=sh600

# 从指定日期下载科创板块
engine 5min --sector=sh688 --start=20240101
```

**5. 下载所有股票**
```shell
# 全量下载（股票+指数共约6000+只，默认从2020-01-01）
engine 5min --all

# 从指定日期下载全部
engine 5min --all --start=20230101

# 增量更新全部
engine 5min --all --update
```

**6. 查看统计信息**
```shell
engine 5min --code=sz300773 --stats
```

输出示例：
```
股票 sz300773 5分钟K线统计:
  记录数: 23656 条
  起始时间: 2024-03-04 09:35:00
  结束时间: 2026-03-17 14:20:00
```

**7. 显示存储路径**
```shell
engine 5min --path
```

#### 存储路径
5分钟K线数据存储在：`~/.quant1x/5min/`

目录结构：
```
~/.quant1x/5min/
├── sh600/          # 上海主板600开头
│   ├── sh600000.csv
│   ├── sh600004.csv
│   └── ...
├── sh688/          # 科创板
│   ├── sh688001.csv
│   └── ...
├── sz300/          # 创业板
│   ├── sz300001.csv
│   └── ...
└── sh000/          # 上证指数
    ├── sh000001.csv
    └── ...
```

#### 数据说明
- **默认开始日期**：2020-01-01
- **日期格式**：YYYYMMDD（如 20240101）
- **时间精度**：5分钟（每交易日48条：09:35-15:00）
- **自动复权**：前复权（基于除权除息数据）
- **增量更新**：自动检测本地数据，只下载缺失部分
